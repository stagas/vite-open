## Install

```sh
npm i -g vite-open
```

## Example

```sh
vite-open my-file.ts
```

## Usage

    Usage: vite-open [options] <file>

                     <file>  File to open (can be a .js or .ts file)

       --root=...            Root directory to serve files from
       --https               Use https
       --quiet               Quiet output
